# FinanceHub

Sistema web de controle financeiro pessoal e familiar.

## Objetivo

Ajudar usuários a registrar receitas e despesas, acompanhar o saldo mensal e visualizar os lançamentos recentes de forma simples.

## Funcionalidades da primeira versão

- Cadastro e login de usuários
- Registro de receitas
- Registro de despesas
- Categorias de lançamentos
- Dashboard com saldo, receitas e despesas do mês
- Histórico de lançamentos

## Telas planejadas

- Login
- Dashboard
- Novo lançamento

## Tecnologias planejadas

- Java
- Spring Boot
- PostgreSQL
- React
- Git e GitHub
